# Linux SysAdmin Labs

Repositório destinado ao estudo prático de administração de sistemas Linux utilizando Ubuntu Server em ambiente virtualizado.

O objetivo deste projeto é documentar laboratórios realizados durante minha jornada de aprendizado em Linux, Infraestrutura, Administração de Sistemas e Segurança da Informação.

## Tecnologias Utilizadas

* Ubuntu Server
* VirtualBox
* Git
* GitHub
* Bash
* Linux CLI

## Objetivos

* Aprender administração de usuários e grupos.
* Gerenciar permissões de arquivos e diretórios.
* Automatizar tarefas com Bash Script.
* Administrar serviços Linux.
* Trabalhar com logs e monitoramento.
* Desenvolver fundamentos para SysAdmin, SOC e Cyber Security.

## Laboratórios

### Lab 01 - Usuários, Grupos e Permissões

Criação de ambiente corporativo simulado utilizando usuários, grupos, departamentos e controle de acesso baseado em permissões Linux.
