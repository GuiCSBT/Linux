# Lab 01 - Usuários, Grupos e Permissões no Linux

## 🎯 Objetivo
Simular um ambiente corporativo utilizando Ubuntu Server para praticar o gerenciamento de usuários, grupos, diretórios e permissões.

## 🏢 Cenário
Criação de uma estrutura empresarial fictícia chamada **Pantuni** para representar os departamentos e colaboradores dentro de um servidor Linux.

### Estrutura da Empresa
* Compras
* Contabilidade
* Financeiro
* Gerência
* Marketing

---

## 🛠️ Estrutura Criada

### 👤 Usuários
* `paulo.pantuni`
* `paulo.pantuni.jr`
* `rita.pantuni`

### 👥 Grupos
* `Pantuni_Gerente`
* `Pantuni_Financeiro`
* `Pantuni_Marketing`
* `Pantuni_Compras`
* `Pantuni_Contabilidade`

---

## 💻 Atividades Realizadas

| Ação | Comando Utilizado | Descrição |
| :--- | :--- | :--- |
| **Criação de usuários** | `useradd` | Criação das contas de usuários do sistema. |
| **Criação de grupos** | `groupadd` | Representação dos departamentos da empresa. |
| **Associação de usuários** | `usermod` | Vinculação dos usuários aos seus respectivos setores. |
| **Criação de diretórios** | `mkdir` | Criação da estrutura de pastas representando os departamentos. |
| **Controle de permissões** | `chown` e `chmod` | Restrição de acesso aos recursos corporativos. |

---

## 📚 Conceitos Aprendidos
* Gerenciamento de Usuários e Grupos no Linux
* Estruturação de Diretórios
* Propriedade de Arquivos
* Controle de Acesso Baseado em Grupos (Isolamento de Privilégios)

## ✅ Resultado
Ao final do laboratório, foi possível criar uma estrutura corporativa funcional, compreendendo na prática como o sistema Linux realiza o controle de acesso e protege as informações em ambientes empresariais.
